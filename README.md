Q: 强化学习目标是指网络吗？规划问题怎么表示？
现在在规划问题里面有两个问题是可以表达的，就像切箱问题和 TSP 问题，因为它可以去表达成一个序列。它在 Reinforcement Learning 也很好表达，对于 TSP 而言就是线路的一个长度；对于切箱问题而言，就是箱子的表面积。切箱问题和 TSP 问题，输入以商品大小和点的坐标表达，reward 用长度和面积表达。
